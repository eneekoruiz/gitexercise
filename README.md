# gitexercise

An exercise for practicing with GitHub and IntelliJ.

## Description

This is a simple JavaFX application created as a learning exercise for practicing Git workflows (branching, merging, pull requests) and working with IntelliJ IDEA.

## Tech Stack

- **Language:** Java
- **UI Framework:** JavaFX 17
- **Build Tool:** Maven

## Prerequisites

- Java 11 or higher
- Maven

## How to Run

```bash
mvn clean javafx:run
```

## Project Structure

```
src/main/java/eus/ehu/gitexercise/
├── HelloApplication.java    # Main application entry point
└── HelloController.java     # Controller for the UI
```
