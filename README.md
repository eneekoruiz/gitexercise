# JavaFX Git Exercise

Introductory JavaFX exercise focused on Git, Maven, and basic MVC structure.

## What it covers

- version control setup
- Maven build basics
- JavaFX entry point, controller, and view wiring
- Java module configuration

## Architecture

The exercise uses an FXML view and a JavaFX controller for its small interaction flow. It has no persistence or service layer; the remaining files demonstrate project structure and Git workflow rather than a multi-layer application.
